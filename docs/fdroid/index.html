RequestTracker F-Droid Repository

This is the F-Droid repository for the RequestTracker Android application.

Add this repository to F-Droid

To add this repo to your F-Droid client, use the following repository URL:

https://sharpninja.github.io/RequestTracker/fdroid/repo

Steps:

  1. Open F-Droid on your Android device
  2. Go to Settings → Repositories
  3. Tap the + button to add a new repository
  4. Paste the repository URL above
  5. Tap OK to add the repository
  6. Pull to refresh and search for "RequestTracker"

About RequestTracker

RequestTracker is an Avalonia-based Android app for browsing, searching, and analyzing Copilot session logs. It supports phone (portrait NavigationView) and tablet (desktop-like) layouts.
